# AI Tab Navigator: A Tab Hoarder's Best Friend

## The Story Behind the Extension

As a student, my biggest shame is that I can never seem to control my tabs. First, I start off with one tab in a clean blank window...zen..but then it easily fills up with 2, 3, 4 and many more tabs than i can count. When I realized that I had officially become a **tab-hoarder**, it was crucial that I seek solutions before I drown in this chaos. Instead of closing my tabs like a sane person would, I decided to create AI tab navigator.

## What exactly is AI Tab Navigator?
It is a Chrome extension that relies on Chrome's built-in AI APIs to help tab-hoaders around the world (who speak English) to search for tabs using natural language. Let's see it in action:

## What Problems does AI Tab Navigator solve?

Traditional tab management doesn't work:
- **Browser search** only finds tabs by title (useless when you have 20 tabs with "Anime" in the name or you just don't remember the title)
- **Tab groups** require manual organization (who has time for that?)
- **Bookmarks** become another graveyard of links you'll never revisit
- **Close all tabs?** Too scary. What if you need one of them?

Sometimes, what we need is a tool that accepts our chaos just rolls with it.

### The Technical Magic

The extension uses a three-stage AI verification process to prevent hallucinations:

1. **Initial Assessment**: Does this tab relate to the query?
2. **Challenge Reasoning**: WHY does it match? What specific text proves it?
3. **Prove It**: Go back to the data and quote the exact evidence

This ensures the AI never makes up matches - it must prove every result with quoted text from the actual tab.
